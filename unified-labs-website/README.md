# Unified Labs Website

This is a modern, high-performance corporate website built with React, Vite, Tailwind CSS, and Three.js.

## Getting Started

### 1. Installation

After unzipping the project, navigate to the project directory and install dependencies:

```bash
cd unified-labs-website
npm install
```

### 2. Development

Run the local development server:

```bash
npm run dev
```

Visit `http://localhost:5173` to see the site.

### 3. Production Build

To build the project for production:

```bash
npm run build
```
